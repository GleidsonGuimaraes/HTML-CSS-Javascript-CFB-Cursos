Links e formatação CSS para links

Para criar-mos um link utilizamos a tag 'a'.
Podemos criar links que nos direcionam a outras páginas
de internet, de download e os links conhecidos como âncoras.
Os links âncoras servem para facilitar a navegação dentro de uma página.
