Tags html, head e body

A tag html engloba todas as outras tags da página web.

A tag head serve para colocar-mos as configurações e definições da página. A maior parte desse conteúdo permanece implícito ao usuário.

A tag body contém todo o conteúdo da página. Aqui vai conter todos os elementos que o usuário vai interagir.

Conheça sobre as tags br, h e p

Tag br

As tags br servem para realizar
uma quebra de linha no texto.

Tags h

As tag h possui uma referência que vai do 1 ao 6, onde 1 possui uma maior fonte e 6 uma menor.
São tags utilizadas para definir títulos e subtítulos.

Tag p

A tag p serve para definir um parágrafo dentro do da página. Sempre que uma tag p é encerrada no documento html, ele executa uma quebra de linha sem a necessidade da tag br.
