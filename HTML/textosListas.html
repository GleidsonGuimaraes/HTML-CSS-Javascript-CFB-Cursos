Listas

Em Html, temos dois tipos de listas. As não ordenadas, que são representadas pela tag ul, e as ordenadas, que são representadas pela tag ol.

Para cada item colocado dentro de uma das listas, você deve utilizar a tag li. Essa tag não necessita ser fechada.

Podemos mudar o padrão de exibição das listas utilizando o parâmetro type.

Exemplo de lista não ordenada

  • Vermelho
  • Verde
  • Azul
  • Amarelo
  • Preto
  • Branco
