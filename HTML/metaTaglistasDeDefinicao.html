Meta tag e listas de definição

As listas de definição não possem indicadores para os elementos e são representadas pela tag dl.
Elas são compostas por uma definição de título representada pela tag dt, e uma definição de descrição representada pela tag dd.
Observe que a formatação dessa tag já te fornece uma margem para separar o título da descrição.

Meta Tags

As metas tags são tags de configurações e definições do site e são utilizadas dentro da tag head.
Essas tags são implícitas ao usuário final.
